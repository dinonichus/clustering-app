1) Clone this repository
2) Create & activate virtual env:
   ```bash
   python -m venv .venv
   # Windows:
   .venv\Scripts\activate
   # macOS/Linux:
   source .venv/bin/activate
   ```
4) Dependencies:
   ```bash
   pip install -U pip
   pip install -r requirements.txt
   ```
5) Run streamlit
   ```bash
   streamlit run /app/App.py
   ```
